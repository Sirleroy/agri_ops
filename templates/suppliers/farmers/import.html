How to Import

  1. Download the CSV template below, or export your farmer data from your existing system (Excel, Google Sheets, Notion, etc.).
  2. Fill one row per farmer. first_name is required; everything else is optional, but more complete data means stronger compliance evidence.
  3. Upload the file directly as .xlsx or save as .csv. Both formats are accepted. If your spreadsheet has multiple sheets, we import the first one — move your data there if needed.
  4. Review the result. You'll see three buckets — Created, Auto-corrected (skim to confirm corrections look right), and Rejected (with per-row reasons). Download the rejected rows, fix them, and re-upload.
